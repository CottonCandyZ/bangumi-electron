## 这是什么？

这是一个 [bgm.tv](https://bgm.tv) 的桌面客户端，目标是做一个好看又好用的收藏整理工具，包括但不限于动画、游戏、电视剧、音乐、图书的进度管理。

它基于 Electron-vite 和 React。

它可能不仅仅是一个 bangumi 客户端，因为我后续可能会支持收藏 bangumi 以外的东西，比如你的 figures。
